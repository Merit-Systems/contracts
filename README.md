# Merit Escrow

Escrow contract for secure token distributions with signature-based claims.

## Core Contract

[`Escrow.sol`](src/Escrow.sol) - The main escrow contract supporting both repository-managed and direct distributions.

## Building

```bash
forge build
```

## Testing

```bash
forge test
```
